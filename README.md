# TestProject
Проект по автотестированию.

Тест 1 - проверка возможности покупки товара с использованием фильтров (test_buy_product.py):
1) Переходим на сайт магазина;
2) Выбираем в каталоге нужный раздел товаров;
3) Перейдя в раздел товаров, выставляем фильтры (ограничиваем максимальную цену слайдером, кликаем по чекбоксам "Доступен для доставки", "Товары по акции",
   скроллим вниз, кликаем по пункту "Гарантия", выбираем "1 год"), применяем фильтры;
   Выбираем товар среди отфильтрованных товаров, переходим на страницу товара, осуществляем проверку по названию и цене товара;
4) Добавляем товар в корзину, переходим в корзину, осуществляем проверку по названию и цене товара;
5) Переходим к оформлению заказа, кликаем по кнопке авторизации, вводим случайный номер, нажимаем на кнопку получения кода.

Тест 2 - проверка возможности покупки товара с использованием поиска (test_search_buy_product.py):
1) Переходим на сайт магазина;
2) Осуществляем поиск товара по названию, переходим на страницу с результатами поиска;
3) Выбираем товар, переходим на страницу товара, осуществляем проверку по названию и цене товара.
4) Добавляем товар в корзину, переходим в корзину, осуществляем проверку по названию и цене товара.
5) Переходим к оформлению заказа, кликаем по кнопке авторизации, вводим случайный номер, нажимаем на кнопку получения кода.

Тест 3 - проверка покупки сразу трех товаров с использованием фильтров (test_buy_3_products.py):
1) Переходим на сайт магазина;
2) Выбираем в каталоге нужный раздел товаров;
3) Перейдя в раздел товаров, выставляем фильтры (ограничиваем максимальную цену слайдером, кликаем по чекбоксам "Доступен для доставки", "Товары по акции",
   скроллим вниз, кликаем по пункту "Гарантия", выбираем "1 год"), применяем фильтры;
4) Добавляем товары в корзину, переходим в корзину, проверяем, что суммарная цена товаров равна итоговой цене в корзине;
5) Переходим к оформлению заказа, кликаем по кнопке авторизации, вводим случайный номер, нажимаем на кнопку получения кода.


На всех этапах проводится проверка на соответсвие URL и ключевого слова на странице.

В метод сравнения названия товара добавил исключение, так как название в разделе товаров немного отличается от названия на страничке товара и в корзине. Несоответствие названий выводится в консоль, не прерывая тестирование.

На странице корзины есть два поля, с ценой товара и суммарной ценой, на содержание корректной цены проверяюся оба даже при покупке одного товара.

При добавлении нескольких товаров в корзину выдержана пауза в 3 секунды, так как если прожать кнопки добавления товаров моментально, в корзину добавится только один из них.
Также в корзине осуществлена сортировка товаров по цвету и цене, поэтому порядок товаров меняется и проверка на соответствие значений цен и названий со значениями в разделе товаров в этом тесте не происходит.

Также на этапе добавления товаров в корзину в половине случаев появляется всплывающее уведомление, с его закрытием тоже поработал.
