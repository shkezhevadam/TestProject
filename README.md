## О проекте

Автотесты UI для проверки ключевых сценариев покупок в интернет-магазине.

**Что проверяем:**
- Покупка товара с использованием фильтров (цена, акция, доставка, гарантия)
- Покупка товара через поиск по названию
- Добавление в корзину сразу трёх товаров и проверка итоговой суммы

**Особенности реализации:**
- Сравнение названий товаров гибкое (различия в названиях на разных страницах не прерывают тест, а выводятся в консоль)
- Автоматическое закрытие всплывающих уведомлений
- Принудительные паузы (3 сек.) при добавлении нескольких товаров для стабильности
- Проверяются оба ценовых поля в корзине (цена товара и итоговая сумма)

## Требования

- Python 3.11+
- Браузер Chrome (последняя версия)
- ChromeDriver (совместимый с вашей версией Chrome)
